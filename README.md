# udacity-mobile-flashcards

This is an app that you can add flashcards, for Udacityâ€™s React Nanodegree. It is a mobile application that allows users to study collections of flashcards. The app will allow users to create different decks of flashcards, then take quizzes on those decks. The app is built with React Native. 

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes

### Installing

* Clone project locally `git clone https://github.com/ddutra9/udacity-mobile-flashcards.git`.
* Go to react app `cd udacity-mobile-flashcards`.
* Install all project dependencies with `yarn install`.
* Run server with `yarn start`.

### Supported Handware
This app has been testing on a Xiaomi mi a2 lite running Android 8.1.0
